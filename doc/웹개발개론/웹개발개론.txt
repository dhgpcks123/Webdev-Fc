<h1>웹이란 무엇인가?</h1>
    WWW W3은 인터넷에 연결된 컴퓨터를 통해 사람들이 정보를 공유할 수 있는 전 세계적인 정보 공간을 말한다.

    Web의 용도는 다양하게 나눌 수 있다.
    Web Site. google, naver, daum, facebook 등 HTML로 구성된 여러 사이트들

    API(Application Programming Interface) 카카오 API, 네이버 API, 금융 API 등등

    User Interface 크롬, 사파리, 스마트 와치, IP TV

    웹을 구성하는 3가지 기본 요소
    URI, HTTP, HTML

    -URI(Uniform Resource Identifier)
        리소스 식별자

        특정 사이트
        특정 쇼핑 목록
        동영상 목록

        모든 정보에 접근할 수 있는 정보
    -HTTP(Hypertext Trasnfer Protocol)
        어플리케이션 컨트롤 get,post,put,delete...
    -HTML(Hyper Text Markup Language)
        하이퍼미디어 포맷
        XML을 바탕으로 한 범용 문서 퐷. 브라우저는 이거 읽어서 사용자가 알아보기 쉬운 형태로 만들어줌.
